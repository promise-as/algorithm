### `03.throttle.html`

- 一个事件在重复执行，比如滚动事件，这样很耗内存。
- 函数节流：过滤掉重复的滚动事件。
- 比如，一个事件是每隔一秒执行一次，在不影响用户体验的情况下，使用函数节流，使这个事件在第一秒的时候执行一次，在第61毫秒在执行一次。
- 间隔多少毫秒执行一次，需要通过测试得出来。

- 高级函数：函数的返回值是函数。
